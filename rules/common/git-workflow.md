# Git Workflow

- Commit format: feat(scope): description / fix(scope): description
- Branch naming: codex/upgrade-YYYYMMDD or claude/[name]
- Dated docs: YYYY-MM-DD_[topic].md
- Never commit API keys, tokens, or secrets
- Never force push to main/master
- Never amend commits unless explicitly requested
- Stage specific files, not git add -A
- Co-author line: Co-Authored-By: Claude Opus 4.6 <noreply@anthropic.com>
