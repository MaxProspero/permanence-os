# Governance Rules

- Prime Directive: Automation can assist, but human authority is final
- 60/30/10 Rule: 60% deterministic code, 30% rule-based logic, 10% AI/LLM
- All consequential actions require human approval
- Spending gate must authorize all API costs
- Compliance gate validates against DNA triad before execution
- Sentinel agent can veto any action that violates constitutional rules
- Draft queue pattern: generate -> voice check -> queue -> human review
- Never auto-publish content without human approval
