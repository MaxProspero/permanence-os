# UI Patterns

## Visual System
- Glassmorphism cards as default card style
- backdrop-filter: blur(8px) for glass effect
- Staggered animations via .animate:nth-child

## Typography (approved fonts only)
- Sora: body text
- IBM Plex Mono: code, data, tables
- Orbitron: display headings, labels
- DM Mono: metadata, timestamps
- FORBIDDEN: Inter, Roboto, Arial, Space Grotesk

## Colors
- --c1 (teal #00e5c4): primary accent, bullish
- --c2 (gold #efbb5f): secondary accent, warnings
- --c3 (blue #73b8ff): tertiary, info
- --rose (#ff5c8a): danger, bearish
- --violet: accent highlights

## Chart Conventions
- Bullish candles: #00e5c4 (teal)
- Bearish candles: #ff5c8a (rose)
- Grid lines: var(--line) at 0.5px
- 10% padding top/bottom (padY)
