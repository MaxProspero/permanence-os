# FOUNDATION Discovery Call (20 Minutes)

## Call Goal
- Confirm fit, lock scope, select price tier, and move to kickoff.

## Agenda
1. `0-3 min`: Context and current pain.
2. `3-10 min`: Workflow diagnosis (what breaks now).
3. `10-15 min`: Show target operating loop and deliverables.
4. `15-18 min`: Confirm package and timeline.
5. `18-20 min`: Close: invoice + kickoff date.

## Key Questions
- What is currently costing you the most time each day?
- What must be running reliably by day 7?
- Which workflows are most revenue-relevant right now?

## Scope Lock Script
`Based on what you shared, I recommend [Pilot/Core/Operator]. This covers command center, automation, money loop, and governance setup in 7 days.`

## Close Script
`If this works for you, I send invoice now and we start [date]. Once paid, kickoff begins same day.`

## Post-Call Actions
1. Send invoice and kickoff checklist within 30 minutes.
2. Send calendar invite for kickoff session.
3. Start implementation checklist.
