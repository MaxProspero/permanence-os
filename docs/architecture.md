# Architecture

Permanence OS is a governed personal intelligence system built around compression and authority.

## Authority Stack

```
Layer 0: Human (Payton) ─────────── Final Authority
         │
Layer 1: Base Canon ─────────────── Constitutional Law
         │
Layer 2: Polemarch ──────────────── Governor & Router
         │
Layer 3: Executive Bots ─────────── Strategy Translation
         │
Layer 4: Department Bots ────────── Specialized Work
         │
Layer 5: Audit Loop ─────────────── Evolution & Learning
```

## Governance First

- The Canon is read-only to agents.
- Polemarch enforces, routes, and escalates.
- Agents execute bounded roles with strict lane discipline.
