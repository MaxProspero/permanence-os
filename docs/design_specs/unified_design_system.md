# Unified Design System - Personal Intelligence OS
# Finance OS (Light) + Social OS (Dark) + Shared Components

## Typography: Geist Sans (body) + Geist Mono (data) + Satoshi (display/emotional)
## Fallback: Sora, IBM Plex Mono, Orbitron, DM Mono (current approved fonts)

## The One Color Decision: #471CFF violet tints ALL neutrals
## Every bg, border, shadow carries a trace of brand hue

## Fraternal Twin Principle:
## - Same fonts, spacing, radius, component structure
## - Different colorways per module (light studio vs dark command center)

## Croissant Layers: 0-5s glance / 30s bite / deep work center

## Key Tokens (abbreviated - full blocks in user messages):
## Finance light bg: #F9F8FF (violet-tinted off-white)
## Social dark bg: #0C0B12 (violet-tinted near-black)
## Agent active: #00E5A0 (teal-mint)
## Agent pending: #FFD166 (warm gold)

## Border radius: 6/10/14/20/9999px
## Spacing: 4px base grid
## Animation ease: cubic-bezier(0.16, 1, 0.3, 1) - spring feel
## Module switch: 350ms crossfade

## Left-border accent stripe on EVERY card (the signature)
## Numbers ALWAYS in Geist Mono with tabular-nums
## Labels ALWAYS uppercase + letter-spaced
## Agent status bar: 40px, persistent, never scrollable-away
