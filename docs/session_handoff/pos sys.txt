    currency TEXT DEFAULT 'USD',
    approval_status TEXT DEFAULT 'pending', -- pending, approved, rejected

    context_tokens_used INTEGER,
    max_context_window INTEGER, -- Fuel gauge capacity

    last_ping TIMESTAMPTZ DEFAULT NOW()
);

    def run_command_center(self):

        while True:

    def delegate_task(self, task):

        # Decompose intent into sub-tasks via the CEO model

        # Parallel Execution in Sandboxes
        for step in plan['steps']:

3. Critical Implementation Details
* The SSE Bridge: By setting the transport to sse on port 8811, your orchestrator can access tools (like your Obsidian vault, Gmail, or your local filesystem) over the network. This is what allows your cloud-based worker to operate on your local machine as if it were a "hand".

* Secrets Management: Do not hardcode your API keys. Use docker mcp secret set within this environment to store tokens for platforms like ClickUp or Stripe. This keeps them nestled safely inside the Docker environment.
* Persistent State: The workspace volume mapping ensures that when an agent writes a spec.md or tasks.md, the file persists on your local machine, allowing the folder to remain your primary "UI".
