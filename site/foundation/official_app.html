Ophtxn App Studio

Shape your assistant profile and generate your first command run.

This page helps you define what your assistant should optimize for, then gives a ready command plan you can run in terminal or via Telegram.

1) Personal runtime profile

Fill profile fields and click "Generate Command Pack".

2) Official launch checklist

  1. Run your generated profile commands first.
  2. Start local services: `python cli.py foundation-site` and `python cli.py foundation-api`.
  3. Open `http://127.0.0.1:8797/app/ophtxn` and verify session + ops summary.
  4. Run daily cycle in Telegram with `/ops-pack strict` and idea flow with `/idea-run queue=1`.
  5. Publish your first public writeup using the press kit from this site.
